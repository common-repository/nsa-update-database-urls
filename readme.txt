Updates all urls and content links in your website including images and guids, content.

== Description ==

If you move your WordPress website to a new domain name, you will find that internal links to pages and references to images are not updated. This plugin fixes that problem by helping you change old domain urls and links in your website. very easy to use.

= Features: =
*   Users can choose to update links embedded in content, excerpts, or custom fields etc
*   Users can choose whether to update links for attachments in content and media
*   View how many items were updated once user update urls



== How It Works ==

Upload nsa-update-database-urls to /wp-content/plugins/ directory Activate plugin in admin area plugin menu
go to settings > update urls with Nsa 
follow the Instruction

== Plugin Settings ==

Plugin settings can be found in admin area, settings -> update urls with NSA


== Installation ==

Installation and uninstallation are extremely simple. You can use WordPress' automatic install or follow the manual instructions below.

= Installing: =

1. Download the package.
2. Extract it to the "plugins" folder of your WordPress directory.
3. In the Admin Panel, go to "Plugins" and activate it.
4. Go to Settings -> Update URLs with NSA to use it.

= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and delete the files/folder for this plugin.

= Usage: =

Using this plugin is very simple. Once it has been activated, navigate to Settings -> Update URLs with NSA and follow the instructions.
